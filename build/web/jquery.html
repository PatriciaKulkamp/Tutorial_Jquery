Introdução

jQuery é leve, "escrever menos, fazer mais", biblioteca JavaScript. O objetivo do jQuery é torná-lo muito mais fácil de usar JavaScript em seu site. jQuery tem um monte de tarefas comuns que requerem muitas linhas de código JavaScript para realizar, e envolve-los em métodos que você pode chamar com uma única linha de código. jQuery também simplifica muito as coisas complicadas de JavaScript, como chamadas AJAX e manipulação DOM. A biblioteca jQuery contém os seguintes recursos: 

            HTML manipulação / DOM
            Manipulação de CSS
            Métodos de eventos HTML
            Efeitos e animações
            AJAX
            Utilities

Dica: Além disso, jQuery tem plugins para praticamente qualquer tarefa lá fora.

Ajax

jQuery oferece vários métodos para a funcionalidade AJAX. Com os métodos jQuery AJAX , você pode solicitar o texto , HTML , XML, JSON ou a partir de um servidor remoto usando HTTP GET e HTTP Post - E você pode carregar os dados externos diretamente para os elementos HTML selecionados de sua página! 

            $.ajax({
                url: "teste.html", // Arquivo
                context: document.body, // Caminho para carregar a resposta
                success: function(data) { // Apresentar a resposta de sucesso
                    alert("Sucesso!"); 
                }
            });

Seletores

jQuery seletores permitem selecionar e manipular elemento (s) HTML.
jQuery seletores são usados ​​para "encontrar" (ou selecione ) elementos HTML com base no ID , classes, tipos , atributos, valores de atributos e muito mais. É com base nos Seletores CSS existentes , e, além disso , tem algumas próprias seletores personalizados.
Todos os seletores jQuery em começar com o sinal de dólar e parênteses : $ ().

O elemento seletor

O seletor de elemento jQuery seleciona elementos com base no nome do elemento.
Você pode selecionar todos os paragrafos em uma página como esta: 

$("p")

Quando um usuário clica em um botão , todos os paragrafos ficarâo ocultos : 

            $(document).ready(function(){
                $("button").click(function(){
                    $("p").hide();
                });
            });

O seletor por #id

O seletor #id jQuery usa o atributo id de uma tag HTML para encontrar o elemento específico.
Um id deve ser único dentro de uma página , então você deve usar o seletor #id quando você quer encontrar um único elemento, único.
Para encontrar um elemento com um ID específico, escrever um caractere de contenção , seguido do id do elemento HTML: 

$("#test")

Quando um usuário clica em um botão , o elemento com id = "teste" será escondido : 

            $(document).ready(function(){
                $("button").click(function(){
                    $("#test").hide();
                });
            });

Formulários

Jquery disponibiliza métodos para facilitar a dinâmica em formulários.
Validações, seleção de elementos, anexo de arquivos, etc.

Exemplos

Para selecionar um botao do tipo submit, e adicionar um evento click: 

            $( "input[type=button]" ).click(function(){
                alert('Clicou no botao de submit');
            });

Selecionar os campos com senha do formulario, e adicionar estilo com borda vermelho. 

            $("input:password").css('border', '1px solid red');
